A memorial and tribute web application for creating and managing digital memorials with dignity.

## Features
- Create and manage memorial pages
- Leave tributes and messages
- Browse memorial listings
- Photo gallery support
- Responsive design

## How to Run
Open `index.html` in any modern web browser.

```bash
# Clone the repo
git clone https://github.com/CodeBreweryZA/UmzukhuluApp.git

# Open directly in browser
cd UmzukhuluApp
start index.html
```

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
